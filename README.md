# Simple Counter App

A simple web-based counter application built using **HTML, CSS, and JavaScript**.  
You can increment, decrement, and reset the counter with a clean user interface.

---

## **Features**

- Increment counter by clicking the `+` button
- Decrement counter by clicking the `-` button
- Reset counter to zero using the `Reset` button
- Simple and responsive design
- Works on desktop and mobile browsers

---

## **Technologies Used**

- HTML5
- CSS3
- JavaScript (Vanilla JS)
